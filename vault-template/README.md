# Public vault scaffold

This directory is the generic starter that `init-vault` copies to a local vault path.

It is intentionally non-personal and safe to publish as a reusable template.

## Included layout

- `00_Inbox/` for capture
- `01_Projects/` for project notes
- `02_Areas/` for ongoing responsibilities
- `03_Resources/` for reference material
- `04_Archive/` for completed notes
- `05_Attachments/` for files and images
- `06_Metadata/` for conventions and seed notes

## GSD convention

See `06_Metadata/Plans/GSD-project-bootstrap.md` for the project layout convention used by this repo.
