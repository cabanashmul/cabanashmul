# Projects

Store project notes here.

For GSD-backed work, use:

- `02_Projects/<ProjectName>/GSD/`
- a `.planning` symlink that points to that `GSD/` directory
- `.git/info/exclude` entries so the symlink stays out of git history

The seed note in `06_Metadata/Plans/GSD-project-bootstrap.md` describes the convention in more detail.
